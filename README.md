# Finances / imersão rocketSeat

## ℹ️ Sobre o projeto

Projeto da rocketseat de Financas, onde voce pode adicionar, e remover transaçoes, filtrando Incomes, e Expenses, e dando um valor total, tudo salvo no LocalStorage do navegador .
